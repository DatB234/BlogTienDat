hướng dẫn

Hugo cung cấp nhiều shortcode mạnh mẽ giúp việc tạo nội dung trở nên dễ dàng hơn. Bài viết này sẽ hướng dẫn bạn sử dụng một số shortcode phổ biến.

Giới thiệu Shortcode

Shortcode trong Hugo là cách nhanh chóng và hiệu quả để chèn các thành phần phức tạp vào bài viết của bạn, chẳng hạn như hình ảnh, video, hoặc bảng biểu mà không cần phải viết mã HTML dài dòng.

  1. Shortcode figure

Shortcode figure được tích hợp sẵn trong Hugo, cho phép bạn thêm hình ảnh kèm chú thích hoặc thuộc tính liên kết dễ dàng. Ví dụ:

Tùy chọn CSS với figure

big: Ảnh sẽ vượt qua giới hạn chiều rộng của khu vực nội dung chính.

left: Ảnh sẽ canh trái.

right: Ảnh sẽ canh phải.

  1. Shortcode highlight

Shortcode highlight được sử dụng để làm nổi bật cú pháp mã nguồn. Bạn chỉ cần chỉ định ngôn ngữ, và Hugo sẽ tự động xử lý.

Ví dụ:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
  1. Shortcode youtube

Bạn muốn chèn một video YouTube vào bài viết? Hugo có shortcode youtube để làm điều này thật dễ dàng.
